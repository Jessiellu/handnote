# 基于云开发的云笔记小程序设计与开发
## Handnote 笔记

==========

### 使用方法

* 小程序无需下载，在微信搜索“Handnote笔记”小程序打开即可使用；
* 笔记界面点击右下角“+”按钮进入编辑笔记界面进行自由编辑；
* 文件界面点击长传按钮选择聊天记录的文件进行上传到云端；
* 打卡界面点击右下角“+”按钮选择创建任务打卡；


==========
